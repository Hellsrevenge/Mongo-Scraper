# Mongo-Scraper
Mongo-Scraper is a web app that uses Mongoose and Cheerio to scrape articles from New York Times website and allows user to leave comments on saved articles.

- Javascript

- Node.js
- Express.js
- Handlebars.js
- MongoDB
- Mongoose
- cheerio NPM Package
